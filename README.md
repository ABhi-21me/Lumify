# ✦ Lumify — Studio Light for Windows

## How to Build Setup.exe

### Step 1 — Install NSIS (one time only)
Download from: https://nsis.sourceforge.io/Download
Install it normally.

### Step 2 — Run build.bat
Double click `build.bat`

That's it! It will:
1. Install PyQt5 automatically
2. Build `Lumify.exe`
3. Package it into `Lumify_Setup.exe`

### Step 3 — Share Lumify_Setup.exe
Anyone can now double click `Lumify_Setup.exe` to install Lumify!
- Installs to Program Files
- Creates Desktop shortcut
- Creates Start Menu shortcut
- Adds proper Uninstaller

## Controls
- **Power Button** — Toggle ring light ON/OFF
- **Space** — Keyboard shortcut for ON/OFF
- **Brightness slider** — Control glow intensity
- **Ring Size slider** — How far inward the glow spreads
- **Warmth slider** — Cool blue to warm orange
- **Corner Radius slider** — Sharp corners to rounded
